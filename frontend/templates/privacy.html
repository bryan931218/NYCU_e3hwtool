隱私權政策

生效日期：{{ effective_date }}

本政策由 {{ legal_entity_name }} 制定，並適用於所有使用 E3 作業追蹤系統（以下簡稱「本服務」）的學生與訪客。

1. 我們收集的資料

  • E3 認證資訊：登入時輸入的帳號密碼或 MoodleSession 值僅用於向 E3 平台取得課程資料。
  • Google OAuth 憑證：僅在使用者選擇日曆同步時，透過授權範圍 {{ google_scope }} 取得存取權杖與更新權杖。
  • 操作紀錄與流量資訊：包含 IP、請求時間、功能使用狀態、匿名統計數據，用於偵錯與濫用偵測。
  • 偏好設定：顯示模式、篩選條件與 Google 同步選項等個人化設定。

2. 資料使用方式

蒐集到的資料僅用於下列目的：

  • 登入 E3 並統整作業資料，以呈現個人化儀表板。
  • 在您允許的情況下，使用 Google API 將作業匯入指定日曆。
  • 偵測異常流量、維護系統安全與品質，以及提供必要的客服協助。

3. 資料存放與保護

  • MoodleSession 與 Google Token 皆儲存在後端加密的持久化儲存體中，不會寫入前端或第三方。
  • 機敏資料與操作紀錄僅限授權維運人員存取，並採取最小權限原則。
  • 若您停止使用本服務，可透過客服信箱請求刪除相關資料。

4. 第三方服務

本服務僅串接下列第三方：

  • 國立陽明交通大學 E3 平台：作業資料來源。
  • Google Calendar API：在授權範圍 {{ google_scope }} 內建立行事曆事件。

我們不會販售或分享個資給其他第三方。

5. 使用者權利

  • 如要撤銷 Google 授權，可在 Google 安全性中心或本服務中解除綁定。
  • 對本政策有任何疑問，可隨時與我們聯繫。
